## Getting started

1. Create a Sitecore Headless website following official [documentation](https://doc.sitecore.com/xp/en/developers/sxa/103/sitecore-experience-accelerator/create-a-headless-tenant-and-site.html).
2. Run the project initialization command in your terminal: 
  ```shell
  npx @astro-sitecore-jss/create-astro-sitecore-jss@latest
  ```
3. Follow the instructions in the terminal to create your project.
4. Start Astro website:
  ```shell
  npm run dev
  ```
